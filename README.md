dotfiles
========

## インストール
```
$ cd
$ git clone https://github.com/shun91/dotfiles.git
$ sh symlink.sh
```

## dotfilesを変更した時
```
$ cd ~/dotfiles
$ git add .
$ git commit -m "commit msg."
$ git push -u origin master
```

## 変更したdotfilesを同期する
```
$ cd ~/dotfiles
$ git pull
```
